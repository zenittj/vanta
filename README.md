# Vanta 
Vanta Bot - Creative Portfolio Website

A premium, single-page portfolio website designed for Vanta, an all-in-one Discord utility bot. This project features a unique "sketchy" and hand-drawn aesthetic, combining high-performance animations with a playful design language.

‚ú® Features

Creative "Sketchy" Design: Utilizes a hand-drawn design system with heavy borders, hard shadows, and the Patrick Hand typeface.

3D Scroll Animations: Implements a custom 3D perspective hero section that reacts to scrolling.

Smooth Scrolling: Integrated Lenis for a buttery-smooth scroll experience.

Interactive Cards: Feature and pricing cards with dynamic hover states and rotation effects.

Dynamic Command List: A searchable list of over 200+ bot commands generated via JavaScript.

Fully Responsive: Optimized "Dock" navigation for desktop and a slide-out menu for mobile devices.

Modern Tech: Built with pure HTML, Tailwind CSS (via CDN), and Vanilla JavaScript for maximum performance.
